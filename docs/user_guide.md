快速入门
首次使用
按照README.md中的步骤安装和运行应用

打开浏览器访问 http://localhost:8501

从左侧导航栏选择功能模块

页面说明
🏠 首页: 平台概览和使用指南

📊 股票分析: 分析单个股票的技术指标

⚖️ 投资组合: 构建和管理多资产组合

📈 风险指标: 计算各类金融风险指标

📋 报告生成: 生成专业分析报告

功能模块详解
1. 股票分析
输入参数
股票代码: 输入股票代码，如AAPL、MSFT

时间周期: 选择分析的时间范围（1月到5年）

数据源: 可选择使用缓存或实时数据

输出结果
价格走势图（K线图）

基本统计指标

收益率分析

技术指标计算

2. 投资组合
创建组合
输入多个股票代码（每行一个）

设置权重分配（等权重或自定义）

选择优化目标

点击"分析投资组合"

分析内容
组合收益率曲线

资产配置饼图

风险贡献分析

相关性矩阵

3. 风险指标
主要指标
VaR (Value at Risk): 在险价值

CVaR (Conditional VaR): 条件在险价值

波动率: 价格波动程度

夏普比率: 风险调整后收益

最大回撤: 历史最差表现

分析方法
历史模拟法

参数法（正态分布）

滚动窗口分析

压力测试

4. 报告生成
报告类型
股票分析报告: 个股详细分析

投资组合报告: 组合表现评估

风险评估报告: 风险指标分析

综合分析报告: 全面分析报告

导出格式
HTML格式（完整报告）

Excel格式（结构化数据）

文本格式（简要摘要）

数据源说明
模拟数据模式
默认使用模拟数据，避免API限制

为不同类别股票生成合理的价格序列

包含OHLC和成交量数据

实时数据模式
可选使用yFinance获取实时数据

注意：有API频率限制

建议配合缓存使用

数据缓存
自动缓存已获取的数据

缓存有效期24小时

减少重复API调用
